# UMKM AI Application for Deployment
## Description
A simple Landing Page and Dashboard to manage RASA configuration to retrain and deploy chatbot models. This is a bundled UMKM AI application for deployment.
## Prerequisites
- Docker
- Node.js
- Git
- S3 Bucket
- MySQL Server

## Run App
```
./start.sh
```

## Stop App
```
./stop.sh
```
